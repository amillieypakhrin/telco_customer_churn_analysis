# telco_customer_churn_analysis
# Customer Retention Analysis

## Introduction

This project focuses on analyzing customer retention and churn patterns using the Telco Customer Churn dataset. The objective was to identify factors that influence customer attrition and provide data-driven recommendations to improve customer retention. The project combines data cleaning, exploratory data analysis, SQL querying, and Power BI dashboarding to transform raw customer data into actionable business insights.

* Business recommendations for customer retention

---

## The Process

1. Imported and explored the dataset.
2. Checked for missing values and duplicate records.
3. Corrected data types and prepared the dataset for analysis.
4. Performed outlier analysis on numerical features.
5. Created tenure-based customer segments through feature engineering.
6. Conducted exploratory data analysis to identify churn patterns.
7. Generated business insights using visualizations and correlation analysis.
8. Wrote SQL queries to answer key business questions.
9. Built an interactive Power BI dashboard to visualize retention trends.
10. Developed business recommendations based on analytical findings.
